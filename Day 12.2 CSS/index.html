Diplay:Flex,align -items

To center an element, we use the align-items property to align the item on the cross axis, which in this case is the block axis running vertically

Diplay:Flex,align flex-start

align-items: flex-start; is a property applied to the flex container that dictates how flex items are aligned along the cross-axis within a single line. When align-items is set to flex-start, the flex items are aligned at the beginning of the cross-axis.

Diplay:Flex,align content:flex-end.

This property controls the alignment of multiple lines of flex items within the flex container along the cross-axis. The cross-axis is perpendicular to the main axis (which is determined by flex-direction).
