Diplay:Flex

The display: flex property in CSS initiates the Flexbox layout model for an element. When applied to a parent element, that element becomes a flex container, and its direct children become flex items.

Diplay:Flex,Flex-Wrap

This CSS property is applied to a flex container and specifies whether flex items are forced onto a single line or allowed to wrap onto multiple lines when there isn't enough space within the container.

  • nowrap
  • Wrap
  • Wrap-reverse

Diplay:Flex,Flex-Wrap:no wrap

This nowrap values specifies that the flex items will not wrap.

Diplay:Flex,Flex-flow

The flex-flow property is a shorthand in CSS Flexbox that combines two individual properties:

  • row-wrap
  • row-wrap-reverse
  • row-nowrap
  • column-wrap
  • column-wrap-reverse
  • column-nowrap

This is a shorthand property that combines two individual Flexbox properties: flex-direction and flex-wrap.
